Лабораторная работа: Тестирование REST API

Описание
Тестирование REST API с использованием фреймворка pytest

Файлы проекта
api.py - клиент для работы с REST API
test_api.py - тестовые сценарии (22 тестa)
conftest.py - фикстуры pytest
requirements.txt - зависимости проекта

Установка и запуск
1. Установите зависимости:
pip install -r requirements.txt
